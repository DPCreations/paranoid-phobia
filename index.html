Paranoid Phobia

This application allows you to secure your file transfers, through untrusted networks.

Potential evesdroppers will only get garbarge. The sending party won't even be able to decrypt the file themself.

How does this work?

  • You will start by generating an in-memory key-pair using this app.
  • You will send the provided messaging key to your friend.
  • Your friend will use this app to encrypt his file with the messaging key you generated.
  • He will then send you the encrypted file through any channel. It won't ever touch our servers.
  • You (and only you) will be able to open the file on this page and decrypt it.

Please note!

Nothing is entirely secure. Not this page, not some expensive software product and not even your own hardware.

A list of things that could go wrong (inconclusive)

  • This site source code was changed on the fly by a MiTM attack, by an evil 3rd-party (can be prevented to some degree by using SSL).
  • The source code was changed on the server. Sorry, I can't help.
  • One or more of the libraries used has security bugs that make them vulnerable.
  • One or more of the libraries include backdoors.
  • The messaging key was intercepted and changed on the fly. The files sent back to you needs to be intercepted too, and reencrypted with your real key, for you not to notice.
  • Your computer is compromised. Nothing protects against backdoors on your own system.

As you can see, lots of things can go wrong. But it's still way more secure than using Facebook chat to send your unencrypted files. Always use multiple layers of defense, and feel free to re-encrypt these files with other methods also.
Don't EVER risk your life. Exchange sensitive material offline. Nothing digital can be trusted these days..
